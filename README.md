# Automotive Embedded Systems Projects

## Project 2: UDS Diagnostics over CAN
- UDS protocol (ISO 14229) simulator in Python
- Session management, Security Access (seed/key), DTC fault memory
- Virtual CAN bus communication
- Target: Visteon, Bosch, Continental

## Tools Used
- Python 3, python-can, SocketCAN
- ISO 14229 (UDS), ISO 15765-2 (ISO-TP)
